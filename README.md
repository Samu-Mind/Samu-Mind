## Hey, I'm Samuel 👋

**Frontend Engineer** specializing in **UI & Design Systems** — I build component architectures and interactive animation systems that ship to production.

My focus is the intersection of **React** and **Rive**: state machines, bone rigs, and modular animation layers integrated directly into web, app, and game engine pipelines. I care about interfaces that are both systematic and alive.

- 🔭 Building **[ClientKosmos](https://github.com/IES-Almunia-25-26-PIDAW/client-kosmos-SamuelAyllon)** — freemium SaaS for freelancers (Laravel 12 + React 19 + shadcn/ui)
- 📐 Deep interest in **component API design**, token-based theming, and accessible UI primitives
- 📍 Jerez de la Frontera, Cádiz

### 🎨 Rive — interactive animation in production

I have hands-on experience designing and shipping Rive animation systems — state machines, parallel animation layers, and modular rigs that developers integrate directly without touching the animation logic.

Rive sits at the layer between design tokens and component behavior. State machines replace ad-hoc CSS transitions with auditable, testable interaction logic — the same mental model as a component's prop API.
